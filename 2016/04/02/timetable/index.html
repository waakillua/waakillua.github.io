在我以为所以的工作都完成就等发布到服务器时，大boss才来临。
手机里的界面真是没眼看……
而且不同手机显示的界面不一样，不同浏览器显示的界面也不一样，要哭了好嘛。

但问题既然存在就要解决

  • 远程调试工具caddy.exe
    电脑和手机在同一个网域（连同一个wifi）就可以在手机查看效果啦
  • 添加头部meta标签
    <meta name="viewport" content="width=device-width,initial-scale=1.0, maximum-scale=1, user-scalable=no"/>
  • 将所有的px都改成rem并使用@media
  • css背景固定属性background-attachment:fixed在手机端不起作用
    解决：hack一下
想在哪个标签加背景，可以在它class后:before.
body:before {
      content: ' ';
      position: fixed;
      z-index: -1;
      top: 0;
      right: 0;
      bottom: 0;
      left: 0;
      background: url(path/to/image) center 0 no-repeat;
      background-size: cover;
}
  • 一个更换背景图片的皮肤按钮（position:fixed;）在微信里打开木有显示，但是用魅族自带浏览器打开可以显示，舍友手机微信打开也可显示。
